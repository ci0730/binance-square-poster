网络代理

访问币安需网络可达。软件会自动读取 Windows 系统代理；也可手动填写。账号若未单独配置代理，将统一使用下方设置。

全局代理设置
在「账号管理」中选择「使用全局代理」的账号，发帖与拉取历史均走此处配置。若梯子已开启系统代理，可留空主机与端口，软件会自动读取。

浏览器设置

拉取广场历史、评论等功能需启动浏览器。安装版已内置 Chromium，一般无需填写；若异常，可指定本机 Chrome 或 Edge 路径。

常见路径：Chrome — Program Files\Google\Chrome\Application\chrome.exe；Edge — Program Files (x86)\Microsoft\Edge\Application\msedge.exe
