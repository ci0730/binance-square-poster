首次使用：配置网络代理

本软件需要连接 binance.com 才能发帖。国内网络通常无法直连，请先确保梯子已开启，再完成以下任一步骤。

已检测到 Windows 系统代理

手动配置

  1. 打开梯子软件（Clash / V2Ray 等），确认本地代理端口（常见 7897 或 10808）
  2. 在设置页选择 HTTP 或 Socks5，主机填 127.0.0.1，端口填梯子端口
  3. 保存后点击「测试网络」，成功即可添加账号
